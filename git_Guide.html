git Guide - vscode에서 git 사용하기

local 에서 git 생성

      github 에서 레파지토리
      github 에서 Quick setup 참조하여
      vscode 터미널에서 git remote add origin https://github.com/leouiux/git_vscode.git 실행
      vscode 터미널에서 git push -u origin master 실행

github 에서 clone 받기

      local에 폴더 생성 및 폴더 열기
      터미널을 열어서 git clone http://10.125.251.12/benepia-front/mercury.git
      vscode 터미널에서 git remote add origin https://github.com/leouiux/git_vscode.git 실행
      vscode 터미널에서 git push -u origin master 실행
